Build a dynamic Pokémon Web App using React and the PokéAPI, allowing users to explore, search, and view detailed information about different Pokémon in real time. The application features efficient API integration for fetching live data, a responsive card-based UI for displaying Pokémon details, and robust error handling to ensure a smooth user experience. Users can search Pokémon instantly, enhancing interactivity and usability.

This project was developed as a mini project during my Web Development course, where I applied core concepts such as state management, component-based architecture, and asynchronous data handling, significantly strengthening my skills in building modern, interactive web applications.
